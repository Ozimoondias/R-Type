## Usage
Use Z,Q,S,D for move and ENTER for fire... **obviously**
<br>
<br>
<br>


## Dependencies
* only SFML
<br>


## Compile and run
Compile with a C++11 compliant compiler:
```sh
mkdir build
cd build
cmake ..
make
./cpp_rtype
```
